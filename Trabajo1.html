📝 Ejemplos Resueltos

Ejemplo 1: x² - 5x + 6 = 0

Solución paso a paso:

  • Identificamos: a=1, b=-5, c=6
  • Discriminante = 25 - 24 = 1
  • x = [5 ± √1]/2 = [5 ± 1]/2
  • Resultado: x₁ = 3, x₂ = 2

Ejemplo 2: x² - 4x + 3 = 0

Solución paso a paso:

  • Identificamos: a=1, b=-4, c=3
  • Discriminante = 16 - 12 = 4
  • x = [4 ± √4]/2 = [4 ± 2]/2
  • Resultado: x₁ = 3, x₂ = 1

Ejemplo 3: x² - 3x - 4 = 0

Solución paso a paso:

  • Identificamos: a=1, b=-3, c=-4
  • Discriminante = 9 + 16 = 25
  • x = [3 ± √25]/2 = [3 ± 5]/2
  • Resultado: x₁ = 4, x₂ = -1
